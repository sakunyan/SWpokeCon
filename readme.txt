Swith Controller v1.0
*************************************
Ordinary command: ただ普通にコマンドを送信します。送信時間はおよそ0.1秒です
    ----------------------------------
    Command
        Exception
            LY_MIN : U
            LY_MAX : J
            LX_MIM : H
            LX_MAX : K
        other  : just input commmand!
    ----------------------------------
    Ex. a
        ”A"キーを送ります。
**************************************
HOLD: 任意の時間キーをホールドします。
    ----------------------------------
    Command
        HOLD key time
    ----------------------------------
    Ex. HOLD LX_MAX 3
        上キーを３秒送信します。
***************************************
READ: テキストファイルに記述されているコマンド列を順に実行することができます。
    -----------------------------------
     Command
        READ filename
    -----------------------------------
    Ex. READ test.txt
        test.txtというファイルを読み込んで、上から順に実行します。
***************************************
RECORD: このコマンドを実行したあとのコマンドをSTOPコマンドを入力するまで全て記録します。STOP後にテキストファイルとして保存することができます。
    -----------------------------------
    Command
        RECORD
        STOP
    -----------------------------------
    Ex. RECORD
        a
        b
        a
        STOP
        test.txt
        text.txtファイルにa, b, aのコマンドが記録されます。
***************************************
LEAP: オリジナルコマンドです。旧時渡りすることができます。巣穴にねがいのかたまりを投げ込んだ状態かつ、巣穴からワットを回収していない状態かつ、Aボタンで回収できる位置に立った状態でソフトを落とし、ポケモンにカーソルを合わせてからこのコマンドを実行します。
    -----------------------------------
    Command
        LEAP day
    -----------------------------------
    Ex. LEAP 3
        ３日時渡りします
***************************************
RESET: オリジナルコマンドです。ゲームを終了し、日付を元に戻します。
    -----------------------------------
    Command
        RESET
    -----------------------------------
    Ex. RESET
        日付を１日に戻して、ゲームを終了します。
***************************************
AGAIN: オリジナルコマンドです。ゲームを終了した上で、任意の回数時渡りをします。
    -----------------------------------
    Command
        AGAIN 3
    -----------------------------------
    Ex. AGAIN 3
        日付を１日に戻してゲームを終了し、新たにゲームを起動し３回時渡りをします。
***************************************
FASTTIMELEAP: オリジナルコマンドです。新時渡りをすることができます。ランクマ必須です。ポケモンセンターなど建物の中に移動してから実行するようにしてください。
    -----------------------------------
    Command
        FASTTIMELEAP day
    -----------------------------------
    Ex. FASTTIMELEAP 1000
        1000日分新時渡りをします。１時間でおよそ3000程時渡り可能です。
***************************************
